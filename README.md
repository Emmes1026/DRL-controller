# DRL-controller
KiCad schematic for a custom PCB that automatically turns on daytime running lights when the car starts.

# Project Overview
This repository contains the KiCad schematic and PCB layout for a custom Automatic Daytime Running Lights (DRL) controller. It is a personal DIY project designed to automatically turn on the car's DRLs when the vehicle is turned on, without the need for manual switching.

# How It Works
The circuit is designed to detect when the car is running and activate the lights accordingly.
